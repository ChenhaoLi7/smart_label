# iOS 16.1+ 安装 PWA 详细指南

## 📱 iOS 16.1 的 PWA 支持

iOS 16.1 完全支持 PWA，但"添加到主屏幕"选项的位置可能不太明显。

## ✅ 详细安装步骤

### 方法1: 通过分享菜单（标准方法）

1. **打开分享菜单**
   - 点击浏览器底部的分享按钮（方框+向上箭头）
   - ✅ 你已经完成了这一步！

2. **向下滚动分享菜单**
   - 在当前的分享菜单中，**继续向下滚动**
   - 滚动到底部，找到"添加到主屏幕"选项
   - 这个选项通常在：
     - "书签"选项下方
     - 或者"所有标签页"下方

3. **点击"添加到主屏幕"**
   - 会打开一个新页面
   - 显示应用图标和名称

4. **确认添加**
   - 可以修改应用名称（默认是"Smart Warehouse"）
   - 点击右上角的"添加"按钮

5. **完成！**
   - 应用图标会出现在主屏幕上
   - 点击图标即可全屏运行

### 方法2: 通过书签（备用方法）

如果分享菜单中没有"添加到主屏幕"：

1. **先添加到书签**
   - 在分享菜单中点击"添加到书签"
   - 保存书签

2. **从书签添加到主屏幕**
   - 打开书签
   - 长按书签
   - 选择"添加到主屏幕"

### 方法3: 使用 Safari 菜单

1. **点击地址栏左侧的分享按钮**
   - 不是底部的，是地址栏旁边的

2. **向下滚动**
   - 找到"添加到主屏幕"

## 🔍 为什么可能看不到"添加到主屏幕"？

### 可能的原因：

1. **需要继续向下滚动**
   - iOS 分享菜单可能很长
   - "添加到主屏幕"通常在最后

2. **Safari 设置问题**
   - 检查：设置 → Safari → 高级
   - 确保"网站图标"已开启

3. **网站未识别为 PWA**
   - 需要 HTTPS（你已经有了）
   - 需要 manifest.json（已经配置）
   - 需要正确的 meta 标签（已经配置）

4. **iOS 版本问题**
   - iOS 16.1 应该完全支持
   - 如果版本较低，可能需要更新

## 💡 快速检查清单

在分享菜单中，你应该能看到：
- ✅ 共享
- ✅ 添加到"书签"
- ✅ 将书签添加至...
- ✅ + 新建标签页
- ✅ 新建无痕浏览标签页
- ✅ 书签
- ✅ 所有标签页
- ⬇️ **继续向下滚动** ⬇️
- ✅ **添加到主屏幕** ← 在这里！

## 🎯 如果还是找不到

### 尝试这些方法：

1. **关闭并重新打开分享菜单**
   - 点击屏幕其他地方关闭菜单
   - 再次点击分享按钮
   - 向下滚动到底部

2. **检查 Safari 设置**
   - 设置 → Safari
   - 确保没有限制网站功能

3. **清除 Safari 缓存**
   - 设置 → Safari → 清除历史记录和网站数据
   - 重新访问网站

4. **使用其他方法**
   - 先添加到书签
   - 然后从书签添加到主屏幕

## 📝 验证 PWA 配置

访问以下 URL 检查配置：
- `https://你的IP:8080/manifest.json` - 应该能看到 JSON 配置
- `https://你的IP:8080/icon-192.png` - 应该能看到图标

## 🚀 安装后的效果

安装成功后：
- ✅ 主屏幕上会出现应用图标
- ✅ 点击图标全屏运行（无浏览器界面）
- ✅ 可以离线访问已缓存页面
- ✅ 像原生 App 一样使用

现在试试在分享菜单中**继续向下滚动**，应该能找到"添加到主屏幕"选项！
