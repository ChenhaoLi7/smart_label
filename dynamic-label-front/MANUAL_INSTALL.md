# 手动安装 PWA 指南

## 📱 在手机上安装（当前最简单的方法）

### iPhone/iPad (Safari)

1. **打开应用**
   - 在 Safari 浏览器中访问：`https://172.16.0.45:8080`
   - 或你的实际 IP 地址

2. **点击分享按钮**
   - 点击浏览器底部的分享按钮（方框+向上箭头图标）

3. **选择"添加到主屏幕"**
   - 向下滚动，找到"添加到主屏幕"选项
   - 点击它

4. **确认安装**
   - 可以修改应用名称（默认是"Smart Warehouse"）
   - 点击右上角的"添加"按钮

5. **完成！**
   - 应用图标会出现在主屏幕上
   - 点击图标即可打开，全屏运行

### Android 手机 (Chrome)

1. **打开应用**
   - 在 Chrome 浏览器中访问：`https://172.16.0.45:8080`

2. **打开菜单**
   - 点击浏览器右上角的三个点菜单（⋮）

3. **选择"添加到主屏幕"**
   - 在菜单中找到"添加到主屏幕"或"安装应用"
   - 点击它

4. **确认安装**
   - 会显示安装确认对话框
   - 点击"安装"或"添加"

5. **完成！**
   - 应用图标会出现在主屏幕上
   - 点击图标即可打开，全屏运行

---

## 💻 在桌面浏览器中安装

### Chrome/Edge 浏览器

1. **访问应用**
   - 打开 `https://localhost:8080`

2. **方法1: 地址栏安装图标**
   - 查看地址栏右侧
   - 如果看到安装图标（📥 或 ➕），点击它
   - 点击"安装"

3. **方法2: 菜单安装**
   - 点击地址栏右侧的三个点（⋮）
   - 选择"安装 Smart Warehouse" 或 "安装应用"
   - 点击"安装"

4. **完成！**
   - 会打开一个新窗口（无浏览器地址栏）
   - 这就是 PWA 模式

### Firefox 浏览器

1. **访问应用**
2. **点击地址栏右侧的菜单**
3. **选择"安装此站点为应用"**
4. **确认安装**

---

## 🎯 新添加的安装按钮

我已经在 Dashboard 右下角添加了一个**紫色的安装按钮**（下载图标）！

### 如何使用：

1. **刷新页面**
   - 在手机上刷新应用页面

2. **查看右下角**
   - 应该会看到三个浮动按钮：
     - 蓝色按钮：标签设计
     - 白色按钮：开始扫描
     - **紫色按钮：安装应用** ⬅️ 新添加的！

3. **点击紫色安装按钮**
   - 如果是 Android/Chrome，会直接触发安装
   - 如果是 iOS，会显示安装步骤说明

---

## ✅ 验证安装

安装后，你可以：

1. **查看主屏幕**
   - 应该看到应用图标
   - 图标是你设计的那个（仓库盒子+齿轮）

2. **打开应用**
   - 点击图标
   - 应该全屏运行（没有浏览器地址栏）

3. **检查功能**
   - 所有功能应该正常
   - 可以离线访问已缓存的页面

---

## 🐛 如果还是看不到安装选项

### 可能的原因：

1. **已经安装过了**
   - 检查主屏幕是否已有应用图标
   - 如果已安装，不会再次显示安装提示

2. **浏览器不支持**
   - iOS：必须使用 Safari
   - Android：推荐使用 Chrome
   - 桌面：推荐使用 Chrome 或 Edge

3. **需要 HTTPS**
   - 确保使用 `https://` 访问
   - localhost 可以使用 HTTP

4. **清除缓存**
   - 清除浏览器缓存
   - 重新加载页面

### 解决方法：

1. **手动安装**（推荐）
   - 按照上面的步骤手动安装
   - 这是最可靠的方法

2. **检查 Service Worker**
   - 打开浏览器开发者工具
   - Application → Service Workers
   - 应该显示"已激活并正在运行"

3. **检查 Manifest**
   - Application → Manifest
   - 应该看到应用信息和图标
